AES67 macOS Audio Driver - Installation Information

⚠️ Important: This is experimental software. Please read all information carefully before proceeding.

What This Driver Does

The AES67 Driver creates a 128-channel virtual audio device on your Mac that can send and receive audio over IP networks using industry-standard protocols:

  • AES67 - Audio Engineering Society standard for audio over IP
  • RAVENNA - Professional network audio protocol
  • Dante - Audinate's media networking technology

Installation Location

The driver will be installed at:

/Library/Audio/Plug-Ins/HAL/AES67Driver.driver

After Installation

The driver will appear as "AES67 Device" in:

  • System Settings → Sound
  • Audio MIDI Setup application
  • Your DAW or audio application's device settings
Note: Core Audio will be restarted during installation. Any running audio applications will be briefly interrupted.

Known Limitations

  • NOT tested with real AES67/Dante hardware
  • Hardware compatibility is unknown
  • May not work in production environments
  • Requires macOS 13.0 or later
  • Optimized for Apple Silicon (M1/M2/M3) Macs
  • Intel Mac compatibility is unknown

Configuration

After installation, use the AES67 Manager application (sold separately or available from source) to:

  • Add and configure network audio streams
  • Map stream channels to device channels
  • Monitor stream status and statistics
  • Import/export SDP session files

Uninstalling

To remove the driver, run the uninstall script:

sudo /path/to/uninstall.sh

Or manually remove:

sudo rm -rf /Library/Audio/Plug-Ins/HAL/AES67Driver.driver
sudo killall coreaudiod
