Welcome to AES67 macOS Audio Driver Installer

This installer will install the AES67 network audio driver on your Mac, enabling your system to send and receive audio over IP networks using the AES67, RAVENNA, and Dante protocols.

⚠️ EXPERIMENTAL SOFTWARE WARNING

This is alpha/beta software that has NOT been tested with real AES67 hardware.

By proceeding with this installation, you acknowledge that:

  • This driver may NOT work with real AES67/Dante devices
  • The driver has only been validated with unit tests
  • It has NOT been tested in production environments
  • It may cause system instability or crashes
  • It is NOT recommended for production use
  • You use this software at your own risk

This software is intended for development, testing, and experimentation only.

What Will Be Installed:

  • AES67Driver.driver - Core Audio driver bundle
  • Location: /Library/Audio/Plug-Ins/HAL/
  • The driver will appear as "AES67 Device" in your system

System Requirements:

  • macOS 13.0 (Ventura) or later
  • Apple Silicon (M1/M2/M3) Mac recommended
  • At least 50MB free disk space
  • Administrator privileges required

During Installation:

  • Core Audio will be restarted
  • All audio applications will be briefly interrupted
  • Close any audio applications before proceeding

Click Continue to proceed with installation, or Quit to cancel.
